# Using Docusaurus docs
## Categories
- To create a category (collapsible) simply create a directory
- The category label is the same as the directory name
- Settings such as the category label, position it appears in the sidebar or slug used in the url can be modified in the `_category_.json` file under the newly created directory. Check the [getting-started](./getting-started/_category_.json) for a somewhat useful example. The link in this config is of the `category link` type. Check the [`category link` docs](https://docusaurus.io/docs/sidebar/items#category-link) for more info.


## Documents
- Create a file under some directory
- Check the [front matter options](https://docusaurus.io/docs/create-doc)
